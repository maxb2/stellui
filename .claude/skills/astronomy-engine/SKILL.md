## Dependency

`astronomy-engine-bindings = "2.1.19"` in `Cargo.toml` — raw C FFI bindings generated by `bindgen`. All functions are `unsafe`.

## Full API Reference

See [`astronomy-engine-bindings.md`](./astronomy-engine-bindings.md) for the complete rustdoc-generated API reference (functions, types, constants).

## Naming Convention

- Functions: `Astronomy_PascalCase` (e.g., `Astronomy_Horizon`)
- Types: `astro_snake_t` (e.g., `astro_time_t`, `astro_observer_t`)
- Constants/enum variants: `astro_enum_t_VARIANT` (e.g., `astro_body_t_BODY_SUN`)

## Key Enum Values

- **Bodies**: `astro_body_t_BODY_{SUN, MOON, MERCURY, VENUS, MARS, JUPITER, SATURN, URANUS, NEPTUNE, EARTH, STAR1..STAR8}`
- **Refraction**: `astro_refraction_t_REFRACTION_NORMAL` (atmospheric) · `_NONE` · `_JPLHOR`
- **Aberration**: `astro_aberration_t_ABERRATION` · `_NO_ABERRATION`
- **Equator date**: `astro_equator_date_t_EQUATOR_OF_DATE` · `_J2000`
- **Direction**: `astro_direction_t_DIRECTION_RISE` (+1) · `_SET` (-1)
- **Success check**: `astro_status_t_ASTRO_SUCCESS`

## Usage Pattern

```rust
use astronomy_engine_bindings::{
    Astronomy_MakeTime, astro_observer_t, astro_status_t_ASTRO_SUCCESS, /* etc. */
};

unsafe {
    let time = Astronomy_MakeTime(2026, 3, 11, 12, 0, 0.0);
    let observer = astro_observer_t { latitude: 40.71, longitude: -74.01, height: 0.0 };
    // ... call functions, always check .status == astro_status_t_ASTRO_SUCCESS
}
```

## Coordinate Pipeline

```
catalog Star (J2000 RA/Dec)
  → Astronomy_DefineStar + Astronomy_Equator → equator of date (RA, Dec)
  → Astronomy_Horizon → alt/az for observer
  → hor_to_stereo: r = 2·tan(45° - alt/2), phi = azimuth
  → PolarCoordinates::canvas_orient: phi -= 90° (N=bottom)
  → CartesianCoordinates: x = r·cos(phi), y = r·sin(phi)
  → ratatui Canvas [-2.2, 2.2]; horizon at r == 2.0
```
